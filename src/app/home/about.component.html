Joe Moceri

I'm a Full Stack C#, SQL, Software Developer. I started writing code in 2011 using C first then Python, which was much easier to work with. Using the Python Turtle I drew my first graphic on screen, and I was hooked on coding ever since. Software has always fascinated me, I wanted to write code for video games and eventually I learned how to do this first using Processing (sketches here). I wanted to work with a more popular game engine and I started using Unity3D, which introduced me to C#. From there I began my work as a Web Developer, where I starting working with databases such as SQL Server, MySQL and PostgreSQL. I started working with Amazon Web Services including EC2, their service for message queues (SQS), Simple Email Service, Elastic Beanstalk, RDS, and Lambda. I also began working with Microsoft Azure including Key Vault and WebJobs. After, I spent time working with WebRTC and two integrations with 3rd-party providers samesurf and Twilio for video chat, the ability to text phones using a web browser, and other real-time communications.
