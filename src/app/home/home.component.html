Unity Effects Pack Documentation

AP Effect Base

These effects are components can be applied to 2D and 3D game objects for a variety of transformations. See below.

Base Properties (these are on all effects)

  • Target: This is which game object the component should select.
  • Use This Object: Whether to target the game object this component is on.
  • Random Start Delay: Whether to have a random start delay when the component plays on Start.
  • Random Length: Let the component pick a random length.

On Start

  • Curve: The curve to use that will use it's normalized value to manipulate the effects (ease in / out of corners, straight, etc). Refer to tooltips for how they apply to each effect. NOTE: Curve will be normalized if it isn't between 0 and 1 inclusive.
  • Length: How long the effect should be.
  • Affect Children: Whether the effect should apply to children of the game object (some effects don't support this, like Color)
  • Continuous: Whether the effect should be continuous or not. This can be ping poing or loop.

Events

Events can fire on either when the effect is running. You can have your custom event fire when a certain percentage changes, or when the effect finishes.

  • Action: When to trigger the event. Options are Nothing, Play, and Reverse.
  • Event Type: What type of event. Options are On Change and On Finish.
  • Percentage: At what percentage to trigger the event.
  • E: The set of events.

APColor

APColor can be applied to any game object that has a Graphic, SpriteRenderer or Renderer. It will then use the .color property or the shared materials color property. It can be used to gradient between multiple colors.

Does not support AffectChildren.

  • Colors: An array of colors to pick from
  • Randomize Order: Whether to randomize the order each iteration.

APFade

APFade can be applied to any game object that has a Graphic, SpriteRenderer or Renderer. It will then use the .color property or the shared materials color property. It can be used to fade the alpha on a game object.

  • Curve: 0 is 0% alpha. 1 is 100% alpha.

APMove

APMove can be applied to any game object. It can take any object, give it an end position, and have the object move there.

  • Curve: Determines the strength as a percentage of the movement from 0 to 1.
  • Motion Type: Whether the movement should be absolute or relative.
  • End Position: The end position to move the target to.
  • Movement Type: The type of movement. Options are linear, circular, and sine. Linear means it will simply go there straight with the speed determined from the curve. Circular means it'll move in a circle towards it. Sine means it'll move like a sine curve towards it.
  • Movement Type Sine Oscillations:How many oscillations in the sine movement.
  • Movement Type Sine Amplitude:The amplitude of the sine movement.
  • Movement Type Sine Movement Direction:The direction the movement should be in. Options are positive and negative.
  • Movement Type Circular Starting Angle:At what angle should the movement be.
  • Movement Type Circular Radius:The radius for the circle.
  • Movement Type Circular Turns:How many turns for the circle.
  • Movement Type Sine|Circular Turns:How many turns for the circle.

APRotate

APRotate can be applied to any game object. It will rotate it.

  • Curve: 1 is a 360 positive rotation. -1 is a 360 negative rotation.
  • Pick Random Rotation: Rotate x, y, or z.
  • Pick Random Direction: Rotate forwards or backwards.
  • X: Rotate X.
  • Y: Rotate Y.
  • Z: Rotate Z.

APScale

APScale can be applied to any game object. It will scale it.

  • Pick Random Scale: Pick a random direction to scale in: x, y, z.
  • X: Scale X.
  • Y: Scale Y.
  • Z: Scale Z.

APSquashStretch

APSquashStretch can be applied to any game object. It will apply the animation principle of squashing and stretching using factors of density and stretchiness.

  • Curve: Positive determines the y squash. Negative determines the x squash.
  • Stretchiness: How much it get's pulled apart.
  • Denseness: How much remains after. The thickness of it.
